Practica No #5 Parte 2
Etiquetas utilizadas durante la practica:

1. Etiqueta

<h1>

La etiqueta de título <h1> es muy simple ya que solo modifica el tamaño de la letra para los títulos, la cual se utilizó en más de un título durante la práctica para hacer más llamativo el título en cuestión.

2. Etiqueta

<A name="">

Esta etiqueta por sí misma no hace mucho, ya que sirve para poder darle un nombre dentro del Código a aquello que se encuentre dentro de la etiqueta para que de este modo otras etiquetas (como la siguiente a esta) pueden hacer uso de ese nombre para poder servir como enlaces dentro de la misma página.

3. Etiqueta

<A href="#">

Esta etiqueta trabaja con ayuda de la <A name="">, ya que al momento de usarla, lo que hacemos es poner entre las comillas el nombre que asignamos en la etiqueta anterior y de este modo se creara un vínculo entre ambas que al momento de hacer clic en la parte del texto con esta etiqueta nos llevara hasta la etiqueta <A name=""> sirviendo así como un enlace dentro de la misma página que puede usarse para navegar de manera más ágil.

4. Etiqueta

<img src=””>

La etiqueta necesaria para poder añadir imágenes dentro de la página web, la cual es muy sencilla, ya que dentro de las comillas solo debemos de poner donde se encuentra la imagen, además de eso podemos definir el tamaño de la imagen por medio de "width", "height" y "alt".

5. Etiqueta

<video>

<video> tal como nos dice nos permite añadir un video a la página, al igual que la imagen podemos elegir el tamaño que tendrá el reproductor del video, así como elegir el título del mismo y un "poster" que sirve como una imagen que aparezca en el reproductor mientras este no está reproduciendo nada.

6. Etiqueta

<audio>

Esta etiqueta nos permite añadir un archivo de audio dentro de la página, en ella debemos de especificar cual audio queremos reproducir, eligiendo el título de la canción, los controles para reproducirla, también podemos elegir si esta se reproduce automáticamente al entrar a la página, para luego elegir el archivo en cuestión que usaremos.

7. Etiqueta

<table>

Con ayuda de esta etiqueta es que podemos insertar tablas dentro de la pagina web, podemos elegir el tamaño que tenga la tabla y su alineacion, dentro de esa necesitaremos la ayuda de otras dos etiquetas, <tr> y <td>.

8. Etiqueta

<tr>

Esta etiqueta es la que nos permite crear cada fila dentro de la tabla, de modo que al momento de usarla debemos de poner dentro de ella cada una de las columnas que tendra cada fila.

9. Etiqueta

<td>

Esta etiqueta se usa en conjunto con la de <tr> para darle las filas y columnas deseadas a la tabla que querramos insertar dentro de nuestra pagina web, esta etiqueta se usa dentro de la etiqueta <tr> para juntas hacer las filas y columnas.

10. Etiqueta

<p>

La mas simple de todas y tambien la mas utilizada ya que es la que se usa para introducir parrafos dentro de la pagina, simplemente el texto deseado se pone dentro de la eiqueta para que este sea usado dentro de la pagina. En este tambien se utiliza el <br> para hacer saltos entre los parrafos.
